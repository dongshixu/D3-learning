# commit rule
```
// 提交新功能
git commit -m 'feat: 新功能'
// 修复 bug
git commit -m 'fix: 修复 bug'
// 琐碎提交
git commit -m 'chore: 暂存'
// 重构
git commit -m "refactor: 这是重构"
// 性能优化
git commit -m "perf: 这是重构"
// 样式美化
git commit -m "style: 样式优化"
// revert
git commit -m "revert: 代码回滚"
// 文档
git commit -m "docs: 文档"
```
